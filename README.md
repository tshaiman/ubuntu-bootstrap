# ubuntu-bootstrap

1. After opening a new shell for the first time on your Ubuntu 22.04 , simply run ./install.sh

2. Tool installation contains :

- ZSH
- OH-MY-ZSH with powerlevel10k
- Azure CLI
- Kubectl, helm , kubelogin
- jq, yq, curl, git, snap, essentials
- homebrew for linux
- stern
- envsubst
